REAL CUSTOMERS REAL STORIES

PROJECT NAME
Electronic Boards with DB

OPPORTUNITY
The production line was unable to get real time information on the amount of batteries being produced. The shift had to be over in order to get the complete information on the production line.

SOLUTION
We developed a software to communicate with an Allen Bradley PLC so we could have real time information on the number of batteries produced.

We placed a sensor, stored the information in a database and created a summary to be displayed on an electronic board, as well as in an intranet software, so the status of the line could be visual.

We also developed trend charts to predict downtime so the production line workers could be informed of the real time status and could increase the production in order to get the daily, weekly or monthly goals.

PROJECT NAME
13 CNC Machines and 1 Tool Control.

OPPORTUNITY
The Overall Equipment Efficiency (OEE) was unknown in the machining centers and the downtime was close to 40%

SOLUTION
We created a “black box” to communicate with the Computer Numerical Control (CNC) and get the real time values from several important variables, like (Alarms, In Cycle, Power On, Feed Hold, Speed, etc.). With these values we were able to determine if the machine was working or not.

We also created an internal communication center with the others CNCs and with Tool Room, so every time the CNC operator needed a special tool, instead of stopping the machine and going to pick it up, he only needed to send a message and tool room crew could bring it to him.

After 6 months of implementation the productivity increased more than 50%.

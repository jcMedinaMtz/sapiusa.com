PROJECT NAME
Real Man-Hours

OPPORTUNITY
The man-hours each worker invested on the production line was manually inputted at the end of each shift. This caused uncertainty about the veracity of the data and no data was available until the end of the shift.

SOLUTION
We developed a mobile app to allow supervisors to input the actual man-hours and to take pictures of the status of the products. This information goes directly to a database and it allows the company to have real time information and to have certainty about the veracity of the data.
